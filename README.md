# L UI介绍

#### 1. UI库介绍

> L UI是我为了提升自己对Vue的各种API的熟悉程度和组件化开发的理解而制作的一个UI组件库。

#### 2. L UI特点

> 常用组件的制作
> 使用了Vue的各种API
> 为了提高易用性组件的一些样式和色彩搭配借鉴了业内知名UI组价库 `Ant Design` 和 `Element UI`

#### 3. 特别提醒
使用本框架前 请在 css 中启用 bordder_box

```
*{
    box-sizing:border-box;
}
*::before{
    box-sizing:border-box;
}
*::after{
    box-sizing:border-box;
}
```
